# BrainBalance
full-stack machine learning website that predicts stress levels (Low/Medium/High) and burnout risk based on user lifestyle inputs such as sleep, work hours, mood, and screen time.

## Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Python 

## How to Run
1. Clone the repository
2. Run server:
   node server.js
3. Run model:
   python train.py
4. Open index.html

## Features
- Predict stress level
- User-friendly interface
